(before any work, always git status & git pull) 
	Before you start working...
git status -- check if there are any changes and what branch you are on
git checkout master if you are not on master
git pull -- get latest changes from master before cutting your branch

	Create new PR branch
Git checkout -b my_branch_name   -- creates your own local branch 
Git branch   -- see all  local branches ( your current branch will be in green)
Git branch -a   -- see all local branches & remote branches (in red font) 

	Make Changes/Commit/Push to remote GitHub
Write test or make changes to existing test
git status -- to check all modified file 
git add .  -- add all changed files to the commit 
git commit -m  -- make the local commit and provide description
git push -u origin my_branch_name   -- push your branch to the github

	See local commits and changes
Git log   -- see all commits on this branch ( their hash, author, date, message)
Git show partial hash  -- check changes in files on this particular commit

GitHub PR process 
Create PR   -- ( inside the github interface)
Review PR  ( make necessary changes and repeat steps 5,6,7)
Merge PR

Delete branch on GitHub 
git branch -D my_branch_name --  deletes local branch

How to remove files from your PR you did not intended to commit 
git checkout <pull-request-branch>
git checkout origin/master -- OrderMEUITests/OrderMEUITests.swift
git commit -m "Removed a modified file from pull request" 
git push origin <pull-request-branch>


Discard Changes
git stash
git reset --hard


Delete Branch
git branch -d yourbranch
